# Portfolio — site Jekyll (Minimal theme)

Pour publier :
1. Crée un repo GitHub public (idéal : `tonpseudo.github.io`).
2. Pousse ces fichiers dans la branche `main`.
3. Place ton CV dans `assets/cv.pdf`.
4. GitHub Pages : Settings → Pages → Branch main / root.

Modifier le contenu :
- Les fichiers .md peuvent être édités directement sur GitHub.
